## 操作説明
- マウスで操作
- プレイヤーはマウスカーソルを目指して動きます
- 紫の星型の敵を避けながら、黄色い飴を取っていってください
- 飴を取るたびに敵が増えて、得点は上がります
- ハイスコアを目指しましょう！
